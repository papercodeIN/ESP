## How to add a new video entry

When you add a new video/tutorial, please follow this template to keep the README consistent:

- Section: choose one of `Firmware`, `MQTT Publisher`, `Modbus Client + MQTT Publisher`, `Modbus TCP/IP Server / Device`, `Modbus TCP/IP Client`, or `MQTT Broker`.
- Entry format:
  - Title: Short descriptive title
  - Watch: full YouTube URL
  - Code: full GitHub URL (if code is available)
  - Blog: full blog URL (optional)

Example:

- Title: Publish XYZ Sensor Data to MQTT
  - Watch: https://youtu.be/abcd1234
  - Code: https://github.com/your/repo/path/Sketch.ino

Add the new item to the correct section in `README.md`. If you prefer, submit a pull request and include the links above — I will review and merge.

---

If you'd like, I can also:

- format entries as a Markdown table for compactness,
- automatically scan the repository for matching code folders and fill missing links,
- or generate a small script to validate that each watch link and code link is reachable.

Tell me which of the optional tasks you'd like me to do next.
